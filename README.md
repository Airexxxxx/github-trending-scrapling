# GitHub Trending Scraper 🚀

这是一个自动化抓取 **GitHub Trending** 每日最热门项目的生产力工具。它不仅能精准抓取多维度的项目数据，还能利用 AI 自动翻译项目描述，并生成精美的 Markdown 报告。

## ✨ 核心功能

* **全方位采集**：抓取项目名称、链接、描述、编程语言、总 Star/Fork 数、今日增长量及贡献者信息。


* **AI 智能翻译**：自动调用 `deep_translator` 将英文项目描述翻译为中文，助力快速理解项目核心价值。


* **数据持久化**：支持生成 **JSON 结构化数据**（便于二次分析）和 **Markdown 排版报告**（便于笔记记录与阅读）。



## 🚀 核心优势

* **强隐身抗反爬**：依托 `Scrapling` 库的 `StealthyFetcher`，模拟真实用户行为，有效绕过 GitHub 防御。


* **极高鲁棒性**：采用“游离文本提取”与“XPath 全文搜索”技术，彻底解决因 GitHub 页面结构变动导致的抓取失败问题。


* **开发维护简便**：代码架构采用模块化设计，具备自动兜底机制，面对网页变动仅需微调正则即可快速适配。



## 🛠️ 如何运行

### 1. 环境准备

确保已安装 Python 环境，并安装必要的依赖库：

```bash
pip install scrapling deep-translator

```

### 2. 运行脚本

直接运行主程序即可生成数据文件：

```bash
python github_trending_scrapling.py

```

### 3. 查看结果

运行后，目录下会自动生成两个文件：

* `github_trending.json`：存储原始数据的 JSON 文件。
* `github_trending.md`：排版精美的日报文档，直接打开即可阅读。
